Lambda expression

Introduced in C++11 lambda expression is a callable closure object. Many standard algorithms use lambdas for custom operations e.g std::for_each.

Lambda expression syntax is as follows:

[captures] front-attr (params) specs except back-attr trailing requires contract-specs { body }

Explanation:

captures:
the entities to be captured,
params:
parameter list of operator(),
specs (optional):
can only be specified as mutable - allowed to modify the entity captured by copy and to call object's non-const member functions,
except (optional):
noexcept specifier for operator(),
back-attr (optional):
Attribute specifier that applies to the type of operator(),
trailing (optional):
trailing return type,
body:
the function body.

Lambda expression is a prvalue of a closure type, which means the smallest block scope, class scope, or namespace scope that contains the lambda expression.
